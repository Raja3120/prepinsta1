Projects

School Management system with Jswing and SQL

The School Management System (SMS) project integrates a user-friendly Java Swing-based graphical interface with a MySQL database backend to manage various school operations efficiently. It features modules for user authentication, student and teacher management, class and subject allocation, attendance tracking, examination scheduling, and library management. The system follows the MVC architecture for clean code organization and scalability. It automates routine tasks, centralizes data management, and enhances operational efficiency. This project is ideal for streamlining administrative and academic processes in educational institutions.

Hearth Disease Predictor

The Heart Disease Prediction project leverages machine learning techniques to predict the likelihood of heart disease in patients based on various health parameters. The system uses a dataset of medical records to train models, employing algorithms such as logistic regression, decision trees, or neural networks. The user inputs data through a graphical user interface, and the system outputs a prediction with an associated risk score. This project aims to assist healthcare professionals in early diagnosis and proactive management of heart disease, improving patient outcomes through data-driven insights.
